FocusFlow - Extensão

Esta extensão foi desenvolvida utilizando apenas tecnologias nativas da web (HTML5, CSS3, JavaScript ES6), sem dependências de terceiros ou frameworks.
Instruções de Construção (Build)

Como esta extensão utiliza JavaScript puro, não há processo de transpilação, minificação ou bundling. O processo de "construção" consiste apenas em descompactar os arquivos fonte.
Requisitos de Sistema e Ambiente

    Sistema Operacional: Windows, macOS ou Linux (Qualquer um).
    Ferramentas Externas: Nenhuma. Não é necessário Node.js, NPM ou qualquer compilador.
    Versão do Firefox: Firefox 57 ou superior (Suporte a WebExtensions).

Os seguintes arquivos compõem o código-fonte (nenhum deles é minificado ou gerado por máquina):

    manifest.json
    background.js
    popup.html
    popup.js
    block.html
    block.js
    styles.css
    icon.svg
    README.md (Este arquivo)
